# Changelog

All notable changes to this project will be documented in this file.

This project follows semantic versioning where practical.

## [0.1.0] - 2026-06-08

### Added

- Initial Obsidian desktop plugin for managing self-hosted Memos from a sidebar.
- Settings for Memos URL, access token, default visibility, page size, and language.
- Memo browsing, searching, creating, editing, deleting, pinning, archiving, and restoring.
- Attachment uploads from file picker, paste, drag and drop, and Obsidian vault paths.
